Bienvenido a i-Presence

La aplicación web de Telepresencia de Formación en Centros de Trabajo

Esta aplicación, aún en desarrollo, es para el Proyecto de Administración de Sistemas Informáticos en Red que estoy cursando en el IES Fernando Aguilar Quignón

Está corriendo sobre una microinstancia de Ubuntu Server 12.04 64 bits de Amazon Elastic Compute Cloud (Amazon EC2), gracias a la capa de uso gratuito que ofrece gratuitamente durante un año Amazon Web Services (AWS)

Está desarrollada con Django y para la videoconferencia he utilizado la API Python y javascript de Opentok. Como SGBD uso PostgreSQL

Para la edición del documento colaborativo utilizo Etherpad-lite, el cual necesita tener instalado Node.js
